# 压缩打包目录dist

## 安装说明

`npm install txzip-cli -g`


## 快速使用

```js
  :> txzip-cli name=MM  dir=./dd/  out=publish version=正式
```

## 参数说明: 

| 参数            | 必填   |       类型         |  说明      |
| ----------      | ---- | --------------- | --------- |
| name           |  是    |     String        |  打包出来的名字      |
| dir          |  是    |       String        |  要打包的目录     |
| out          |  是    |       String        |  打包到的目录      |
| version          |  是    |     String          |  版本号      |
